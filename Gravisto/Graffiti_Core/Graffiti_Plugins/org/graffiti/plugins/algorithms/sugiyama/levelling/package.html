Overview

This package contains algorithms for the second phase of the sugiyama-algorithm: the levelling-phaes. In this phase, the nodes have to be put on levels or layers.
Your implementation has to make sure, that the following conditions are satisfied for each node and each edge in the graph (Let A be the source-node of the edge, B the target-node of the edge):

  • When A is on level i, B has to be on level i+1.
    An edge must not point "upwards", i.e. to an upper level, it must not connect two nodes that are on the same level. Additionally, the edges must not span over more than one level
  • You have to save the levels in the object NodeLayers in SugiyamaData. You don't have to worry about the order of an individual layer, the next phase will take care of that
  • There should not be an empty level in the graph

When you have an edge that connects Node A on level i and Node B on level j, where j > i+1, you have to divide the edge into multiple edges. On each layer, you have to insert a dummy-node.

The interface LevellingAlgorithm

Your algorithm has to implement the interface LevellingAlgorithm to be detected as a levelling-algorithm. You have to implement the following methods:

  • Getter/Setter for SugiyamaData
  • check(): Check if your algorithm can handle the current layout of the graph
  • execute(): Compute a levelling of the graph in this method. Save your levelling in NodeLayers in SugiyamaData.
  • setDefaultParameters(): Reset the parameters of your algorithm to its default state.
  • reset(): Reset the internal state of your algorithm

Optional

If it does make any sense for your algorithm, you can add support for animations. If you do add support for animations, the framework will request an animation-object from your implementation instead of calling execute().
Please refer to the documentation of animations for further information.

The abstract class AbstractLevellingAlgorithm

The framework provides a method to add dummy-nodes to the graph. If you extend AbstractLevellingAlgorithm, you can call the method addDummies with a reference to SugiyamaData. This method will insert dummy-nodes for you by looking through NodeLayers in SugiyamaData.
