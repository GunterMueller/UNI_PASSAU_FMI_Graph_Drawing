Menu item "File"

The file menu looks like this:




New: Loads a new frame for a new basegraph.

New View: Opens a new view of the basegraph contained inside active frame.

Open: Opens a graph and its views from a file.

Save: Saves graph and its views.

Save as: Opens save dialog.

Close: Closes active frame.

Exit: Quits program.
