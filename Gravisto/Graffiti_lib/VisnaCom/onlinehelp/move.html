Editing mode "Move"

By pressing the following button one can switch to the "Move"-mode.

In this mode you can select parts of the displayed graph by either left-clicking on them or by dragging the mouse (with left button down) over those parts of the graph. Selected items can be moved by dragging them with the left mouse button. If you use the right mouse button for dragging them, they will become descendants of the node at which you stop dragging.
In this mode you can also move controlpoints of edges. In order to move a controlpoint, select the edge and then drag the controlpoint you want to move with the left mouse button. The pictures below show the effect of moving a controlpoint:
