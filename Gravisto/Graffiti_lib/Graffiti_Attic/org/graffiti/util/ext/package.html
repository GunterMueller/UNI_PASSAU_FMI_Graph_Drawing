Extension methods and extension classes for third-party code including the jdk. Extension methods were (amongst others) proposed by Google as a java language improvement. An extension method is a static method whose first argument is an explicit this-pointer to the object whose class's interface is to be extended. With proper language support its possible to call a method such as 
public static void foo(Bar this_,...)
like this: 
Bar bar = ...;
bar.foo(...);
An extension class does the same thing by subclassing an existing class and adding methods to the interface.
