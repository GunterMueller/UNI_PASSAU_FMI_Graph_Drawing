{ Supported Platforms } At the moment the following configurations are tested:

  • Windows XP using Visual Studio .NET
  • Linux using gcc 3.x, 4.x
  • Solaris using gcc 3.x, 4.x
  • Mac OS X 10.2.x, 10.3.x, and 10.4.x

GTL should basically compile on any platform with an recent gcc, i.e., at least 3.0, although we recommend at least version 3.1. If you suceed in compiling GTL on any platform not listed above, please send us a mail.
